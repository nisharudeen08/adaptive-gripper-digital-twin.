# Adaptive Gripper Digital Twin

## Windows Access (Unity + ROS)
Unity and ROS code are pushed to GitHub, you can access all of it from Windows by cloning the same repo.

On Windows:

```
git clone <your-repo-url>
```

If already cloned:

```
git pull
```

Notes:
- Git tracks your source files, not running ROS/Gazebo processes.
- Large generated folders (for example Unity Library/) should not be in Git.
- Keep secrets (tokens) out of repo; use local config or environment variables.
